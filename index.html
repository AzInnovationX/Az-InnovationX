Instrucciones Detalladas de Pago

1. Abre la aplicación de tu banco.

2. Selecciona la opción para realizar una transferencia (SPEI).

3. Ingresa los datos proporcionados: Banco Azteca, el nombre del titular y el número de cuenta.

4. Ingresa el monto exacto a pagar.

5. Como concepto de pago, por favor ingresa tu nombre completo o número de pedido.

6. Una vez realizada la transferencia, envía una captura de pantalla del comprobante a nuestro WhatsApp para confirmar y procesar tu pedido.
